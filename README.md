📧 Spam Message Classifier
📌 Overview

This is a simple Machine Learning project that classifies messages as Spam 🚫 or Not Spam ✅ using basic text processing and a Naive Bayes algorithm.

🚀 Features
Classifies user input messages
Uses Machine Learning for prediction
Simple and beginner-friendly implementation
Runs in terminal (CLI-based)
🛠️ Technologies Used
Python
pandas
scikit-learn
⚙️ How It Works
A small dataset of messages is created and labeled (spam / not spam)
Text is converted into numerical format using CountVectorizer
A Naive Bayes model is trained on the dataset
The model predicts whether a new message is spam or not
▶️ How to Run
1. Install dependencies
pip install pandas scikit-learn
2. Run the program
python spam.py
3. Enter a message

Example:

Enter message: Win free money
Output: Spam 🚫
📊 Example Output
Input Message	Prediction
Win a free iPhone now	Spam 🚫
Hello, how are you?	Not Spam ✅
⚠️ Limitations
Uses a very small dataset
Accuracy is limited
Not suitable for real-world production use
